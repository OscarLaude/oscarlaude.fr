1. **Prerequisites**
   - Node.js (LTS version)

2. **Installation**
   ```bash
   # Clone the repository
   git clone https://github.com/yourusername/oscarlaude.fr.git
   cd oscarlaude.fr

   # Install dependencies
   npm install
   ```

3. **Development Server**
   ```bash
   ng serve
   ```

   ```bash
   ng build --configuration production
   ```
